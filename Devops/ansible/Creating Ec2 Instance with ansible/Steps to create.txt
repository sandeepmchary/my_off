To do this, we need to override the default Ansibleâ€™s configuration file, Ansible.cfg. It is located by default in /etc/ansible. But, placing a file with the same name in the working directory wil override the default one. Create a new file called ansible.cfg in the current working directory and add the following:
- vi ansible.cfg
	[defaults]
	host_key_checking = False
	private_key_file = /home/ansible/.ssh/<pem file>
