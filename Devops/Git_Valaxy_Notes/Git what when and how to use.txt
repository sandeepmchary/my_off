- git is an open source,distributed version control system designed for speed & effiency
- Work offline (no n/w needed)
	- commiting changes
	- performing a diff
	- Merging branches
	- viewing file history
	- Switching branches
	- obtaining file versions

- difference between git,gitlab,github
	- git is command line
	- Github is web interface
	- Github is also a hosting site
	- free for public repositories
	- Gitlab the open-source github,that enterprises can run on their own severs
	- the command line is the only place you can run all git commands
